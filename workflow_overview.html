当前实现中，所有分镜固定为 10 秒，并在合成前统一执行首帧去除与可选黑场遮罩。
主线阶段顺序与代码保持一致： upstream → asset_extraction → style_bible → asset_prompts → asset_images → storyboard → shot_reference_boards → board_publish → video_jobs → shot_videos → final_video。 可选并行节点为 storyboard_seed，它不会替代正式的 storyboard.json，只作为预分镜规划辅助。
